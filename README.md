## 📊 주요 인사이트

- 한국 방탈출 시장 규모: 연간 1,800억~2,100억 원 (2024년)
- 전국 매장 수: 약 700개
- 글로벌 시장: 2026년 73.7억 달러 → 2032년 137.4억 달러 전망
- 주요 타겟: 20-30대 밀레니얼/Z세대, 기업 팀빌딩
