> **작성일**: 2026년 2월 10일

## 📊 Executive Summary

- **평균 이용 가격**: 17,000원~24,000원/인
- **초기 투자 비용**: 1억~3억 원 (테마룸 2개 기준)

## 2. 주요 수익원

### 2.1 1차 수익: 게임 이용료 (주 수익원)

#### 가격 정책

**시간대별 차등**:
| 시간대 | 가격 | 비고 |
|--------|------|------|
| 평일 낮 | 기본 가격 | |
| 평일 저녁 | +10~15% | |
| 주말/공휴일 | +15~20% | 성수기 할증 |
| 심야 시간대 | 프리미엄 가격 | 특별 운영 |

**인원별 차등**:
- 2인: 가장 높은 인당 가격
- 3~4인: 중간 가격
- 5인 이상: 할인 적용

#### 예약 시스템
- 60분 제한 시간
- 방 회전율 극대화
- 온라인 예약 시스템
- 예약금/선결제 정책

---

### 2.2 2차 수익원

#### 기업 단체 예약
- 팀 빌딩 프로그램
- 워크샵 패키지
- 대규모 단체 할인
- 맞춤형 프로그램 추가 비용

#### 굿즈/머천다이즈 판매
- 테마별 기념품
- 팀 사진 인화

- 방탈출카페로 불리지만 실제 카페 기능은 제한적
- 대기 시간 음료 판매
- 비중: 전체 매출의 3~5% (추정)

#### 특별 이벤트
- 생일 파티 패키지
- 커플 데이트 패키지
- 프로포즈 이벤트
- 추가 비용: 5만~20만 원

#### 협업 마케팅 수익
- 브랜드 협업 테마
- 광고 수익
- 프로모션 파트너십

## 5. 손익분기점

### 5.1 월 매출 데이터

#### 성공 사례 (정주영 대표, 2024년)
- 매장 당 월 매출: **3,000만~4,000만원**

#### 마스터키 브랜드
- **주장**: "매출 50-60% 순이익"
- **ROI**: 1년~2년 내 투자비 회수 가능

---

### 5.2 손익분기점 계산 (추정)

#### 기본 가정
| 항목 | 금액 (원) |
|------|-----------|
| 월 고정비 (임대료+인건비+기타) | 800만~1,200만 |
| 가변비율 | 매출의 20~30% |

#### BEP 계산

**공식**: 
```
BEP 매출 = 고정비 / (1 - 변동비율)
```

**예시 1 (독립 운영)**:
```
고정비: 800만 원
변동비율: 20%
BEP = 800만 / (1 - 0.2) = 1,000만 원
```

**예시 2 (프랜차이즈)**:
```
고정비: 1,000만 원
변동비율: 30% (로열티 포함)
BEP = 1,000만 / (1 - 0.3) = 1,429만 원
```

---

### 5.3 필요 이용객 수 계산

#### 1일 필요 매출
| 월 BEP | 1일 필요 매출 (30일 기준) |
|--------|-------------------------|
| 1,000만 원 | 33만 원 |
| 1,429만 원 | 48만 원 |

#### 1일 필요 팀 수 (팀당 평균 15만 원 가정)
| 1일 필요 매출 | 필요 팀 수 |
|--------------|-----------|
| 33만 원 | 2.2팀 → 3팀 |
| 48만 원 | 3.2팀 → 4팀 |

#### 1일 필요 예약 슬롯
- 테마룸 2개
- 하루 운영 시간: 10시간 (12:00~22:00)
- 1게임 시간: 1.5시간 (게임 1시간 + 준비/정리 30분)
- 1개 룸당 1일 최대: 6~7게임
- 2개 룸 합계: 12~14게임/일

**분석**: BEP 달성을 위해 전체 슬롯의 약 25~35% 예약 필요

## 6. 프랜차이즈 vs 독립 운영 비교

### 6.1 프랜차이즈 운영

#### 장점

| 항목 | 설명 | 가치 |
|------|------|------|
| **브랜드 인지도** | 이미 시장에서 검증된 브랜드 | 초기 마케팅 비용 절감 |
| **운영 노하우 제공** | 교육, 매뉴얼, 운영 지원 | 시행착오 최소화 |
| **테마 공급** | 검증된 고퀄리티 테마 | 테마 개발 비용 절감 |
| **마케팅 지원** | 브랜드 광고, 이벤트 | 통합 마케팅 효과 |
| **네트워크 효과** | 다른 가맹점과 협업 | 규모의 경제 |
| **예약 플랫폼** | 통합 예약 시스템 | 고객 접근성 향상 |

#### 단점

| 항목 | 설명 | 영향 |
|------|------|------|
| **초기 비용 추가** | 가맹비, 교육비, 계약이행보증금 | 초기 투자 500만~2,000만 원 증가 |
| **지속 비용** | 로열티 (3~6%), 광고 분담금, 포스 사용료 | 월 150만~300만 원 추가 |
| **운영 제약** | 본사 운영 가이드 준수 의무 | 창의적 운영 제한 |
| **테마 제한** | 본사 승인 테마만 가능 | 독자적 테마 불가 |
